## Original Chatterbox Acknowledgements

The Chatterbox models build upon:
- **[Cosyvoice](https://github.com/FunAudioLLM/CosyVoice)** - Voice synthesis architecture
- **[Real-Time-Voice-Cloning](https://github.com/CorentinJ/Real-Time-Voice-Cloning)** - Voice cloning techniques
- **[HiFT-GAN](https://github.com/yl4579/HiFTNet)** - High-fidelity vocoder
- **[Llama 3](https://github.com/meta-llama/llama3)** - Language model components
- **[S3Tokenizer](https://github.com/xingchensong/S3Tokenizer)** - Speech tokenization

## This Fork

This FastAPI wrapper adds:
- OpenAI-compatible API endpoints
- Language-specific model variants for OpenWebUI
- Multilingual focus with 23+ languages
- Watermark removal for resource optimization
